Description: The UIElementHandler is a UI system built with the singleton pattern that manages UI elements in your game, including a pause screen overlay that has features for adding/removing scoring elements and end game screens.

This system also includes mobile implementation support for the overlay by including a button for pausing/stats when your game is run on mobile.

This asset includes the following:

  • Pause/Score Overlay
  • Quit Button
  • Separate Overlay Button For Mobile Builds
  • Easily Customizable C# Scripts
  • Documentation for Mobile Implementation
  • Documentation for Adding Elements to the Overlay

Features:

  • Mobile implementation that has separate buttons instead of keypresses
  • Optional code for haptic feedback on mobile devices
  • An overlay displaying scores/points and a quit button
  • Methods that can be called for end game screens and text customizability

Requirements: Unity 2022.3.42f1 or later
